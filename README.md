# Instagram GUI Login - Java Swing

A desktop-based Instagram login replica built using Java Swing and AWT. This project was developed as part of the Final Exam for the Algorithms and Programming course at Universitas Pendidikan Indonesia.

## Features
- **User Authentication**: Secure login using HashMap for credential storage.
- **Sign Up**: Create new accounts with real-time validation.
- **Password Reset**: Interactive pop-up to update user passwords.

## How to Run
1. Clone this repository.
2. Ensure you have Java JDK installed.
3. Compile and run `InstagramLogin.java`.
