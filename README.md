# Instagram GUI Login - Java Swing

A desktop-based Instagram login replica built using Java Swing and AWT. [cite_start]This project was developed as part of the Final Exam for the Algorithms and Programming course at Universitas Pendidikan Indonesia.

## Features
- [cite_start]**User Authentication**: Secure login using HashMap for credential storage[cite: 17, 302].
- [cite_start]**Sign Up**: Create new accounts with real-time validation[cite: 12, 448].
- [cite_start]**Password Reset**: Interactive pop-up to update user passwords[cite: 11, 359].
- [cite_start]**Custom Aesthetics**: Implementation of the iconic Instagram 'Billabong' font[cite: 74, 84].

## How to Run
1. Clone this repository.
2. Ensure you have Java JDK installed.
3. Compile and run `InstagramLogin.java`.
